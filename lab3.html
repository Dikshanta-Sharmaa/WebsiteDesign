About html:

HTML(Hyper Text Markup Language) is the standared language for creating web page.It consists of various types of tag which define the structure and content of a webpage.This lab explain different categories of HTML tags, their, syntax, and their role in web design.

Text formatting tags:

Those tags which is used to change text color, face, size, style, etc is known as text formatting tag.

There are eight types of text formatting tags used in HTML that is given below:

Bold tag

This tag is used to make text bolt in html.

<b> tag is used to make text bold.

eg:<b> Bold text <b>


Itallic Tag

This tag is used to make text itallic format using <i> tag.

eg:<i> Italic text <i>

Underline Tag

This tag is used to make text underline using <u> tag.

eg:<u> Undeline Tag <u>

Delete Tag

This tag is used to make crosss line in the middle of the txt using<del> tag.

eg:<del> Delete tex <del>

Small Tag

This tag is used to make text small using <small> tag.

eg.<small> Small tag <small>

Big Tag

This tag is used to make text big using <big> tag.

eg:<big> Big tag <u>

Sub-script

This tag is used to make text in base using <sub> tag.

eg:H<sub>2 <sub>O

Sup-script

This tag is used to make text in power using <sup> tag.

eg:a+b<sup>2<sup>-2ab;

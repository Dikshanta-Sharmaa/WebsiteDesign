About html:

HTML(Hyper Text Markup Language) is the standared language for creating web page.It consists of various types of tag which define the structure and content of a webpage.This lab explain different categories of HTML tags, their, syntax, and their role in web design.

Use of Image tag in html

The tag which is used to inser/add image in web page is known as <img> tag.

Syntax: <img src=" " alt=" " width=" " height=" ">

Attributes of <img> tag

src(source):The attribute to show image path with image name and extension.

alt(alternate):The attribute used to show text in case of error in source of image tag is called alt attribute.

width:The attribute used to assign image width is called width.

height:The attribute used to assign image heignt.

title:The attribute used to show the short description of image when mouse is hover.

About html:

HTML(Hyper Text Markup Language) is the standared language for creating web page.It consists of various types of tag which define the structure and content of a webpage.This lab explain different categories of HTML tags, their, syntax, and their role in web design.

Table tag:

The tag which is used to insert table inside html is known as table tag. <table> tag is a paired tag.Inside <table> tag we use following tag:
<th>:Table heading(Bold)
<td>:Table data
<td>:Table row
Attributes of <table> tag:
colspan:Colspan is used to merge number of columns. rowspan:Rowspan is used to merge number of rows. border:BOrder attribute is used to make border in table.

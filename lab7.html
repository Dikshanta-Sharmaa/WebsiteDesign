About html:

HTML(Hyper Text Markup Language) is the standared language for creating web page.It consists of various types of tag which define the structure and content of a webpage.This lab explain different categories of HTML tags, their, syntax, and their role in web design.

List:

The arrangement of the element using number, alphabet, symbol, etc in assending or in desending order is called list.

There are two types of list.

1)Ordered list

2)Unordered list

3)Description/Defination list.

Combined/Mixed Nested list

The way of defining ordered list inside another unordered list or unordered list inside order list is called combined/mixed nested list.

For example:

  • Animals
    1. Dog
    2. Rat
    3. Cat
  • Birds
    1. Crow
    2. Sparrow
    3. Parrot
    4. Eagle
  • Flowers
    • White Rose
    • Green Rose
    • Yellow Rose
    • Blue Rose
  • Sports
    1. Football
    2. Circket
    3. Swimming
    4. Running

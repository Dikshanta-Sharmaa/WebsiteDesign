About html:

HTML(Hyper Text Markup Language) is the standared language for creating web page.It consists of various types of tag which define the structure and content of a webpage.This lab explain different categories of HTML tags, their, syntax, and their role in web design.

List:

The arrangement of the element using number, alphabet, symbol, etc in assending or in desending order is called list.

There are two types of list.

1)Ordered list

2)Unordered list

3)Description/Defination list.

Ordered list:The arrangement of element using number and alphabet in assending and desending order is known as ordered list. An ordered list starts with the <ol> tag. Each list starts with the <li> tag. If we don't write or specify any alphabet or number type then its default is marked by numbers.

Syntax:
<ol>
<li> </li>
<li> </li>
......................
</ol>

For eg

An ordered HTML list

  1. Coffee
  2. Tea
  3. Milk

Nested Ordered List:

The way of defining ordered list inside another ordered list is called nested ordered list.

Syntax:
<ol>
<li> </li>
<li>
<ol>
<li> </li>
<li> </li>
</ol>
</li>
<li> </li>
</ol>

For example:

  1. Coffee
  2. Tea
    1. Black tea
    2. Green tea

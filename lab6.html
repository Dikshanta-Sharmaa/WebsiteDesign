About html:

HTML(Hyper Text Markup Language) is the standared language for creating web page.It consists of various types of tag which define the structure and content of a webpage.This lab explain different categories of HTML tags, their, syntax, and their role in web design.

List:

The arrangement of the element using number, alphabet, symbol, etc in assending or in desending order is called list.

There are two types of list.

1)Ordered list

2)Unordered list

3)Description/Defination list.

Unordered list:The arrangement of element using symbol in assending and desending order is known as unordered list. An unordered list starts with the <ul> tag. Each list starts with the <li> tag. If we don't write or specify any symbol type then its default is marked by disc shaped bullets.

Syntax:
<ul>
<li> </li>
<li> </li>
......................
</ul>

For eg

An unordered HTML list

  • Coffee
  • Tea
  • Milk

Nested Ordered List:

The way of defining unordered list inside another unordered list is called nested unordered list.

Syntax:
<ul>
<li> </li>
<li>
<ul>
<li> </li>
<li> </li>
</ul>
</li>
<li> </li>
</ul>

For example:

  • Coffee
  • Tea
    • Black tea
    • Green tea
